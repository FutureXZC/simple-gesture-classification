Member Enumeration Documentation

enum xstream::DisableParam::Mode
strong
Enumerator
PassThrough 

passthrough mode: use inputdata as outputdata; slot size of outputdata and inputdata must be equal;

UsePreDefine 

use pre-difined data as outputdata; slot size of predifine data and outputdata must be equal;

Invalid 

set outputdata status as DataState::INVALID

BestEffortPassThrough 

flexibly passthrough mode

  1. passthrough inputdata to outputdata in order
  2. if slot size of inputdata greater than outputdata, only use the first few slots.
  3. if slot size of inputdata less than outputdata, set status of last few slots invalid.
