2.6. hb_model_verifier 工具¶

hb_model_verifier 工具是用于对指定的定点模型和runtime模型进行结果验证的工具。 该工具会随机生成图片, 进行定点模型推理, runtime模型在模拟器上的推理, runtime模型在板端的推理(如果给定ip可以ping通), 并对其结果进行两两比较, 给出是否通过的结论。

2.6.1. 使用方式¶

hb_model_verifier -c ${config_file} \
                  -a ${board_ip} \
                  -i ${input_img}

2.6.2. 命令行参数¶

hb_model_verifier的命令行参数¶
--cfg-file, -c

hb_model_verifier 配置文件名称。

--arm-board-ip, -a

上板测试使用的arm board ip地址。

--input-img, -i

推理测试时使用的图片。若不指定则会自动生成随机图像做输入。

2.6.3. hb_model_verifier 配置文件¶

# 定点onnx 模型文件
onnx_model: quantized_model.onnx

# hybrid.bin 模型的路径
hybrid_model: hybrid_model.bin

# 模型输入大小
input_shape: 1x224x224x3

# 模型上板时的输入layout
input_layout: NHWC

# 模型上板时的输入类型 (yuv444/rgb(NCHW)/feature)
input_type_rt: yuv444

# 模型训练时的输入layout
input_layout_train: NCHW

# 结果对比精度精确位数
compare_digits: 5

# 输出结果的路径, 请勿使用当前工作路径! 否则会被清空
output_dir: model_verifier_out

2.6.4. 输出内容解析¶

结果对比最终会在终端展示, 工具会对比ONNX模型运行结果, 模拟器运行及上板结果的两两对比情况, 若无问题应显示如下:

Onnx and Sim result Strict check PASSED

Onnx and Arm result Strict check PASSED

Sim and Arm result Strict check PASSED

注意

注意， hb_model_verifier 不支持对已打包的*.bin模型进行验证，否则工作台将产生以下提示：

ERROR pack model is not supported
